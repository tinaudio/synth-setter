# Image Config

`ImageConfig` is the validated config for a Docker image build. Static
fields come from a YAML file under `configs/image/`; runtime fields
(`github_sha`, `issue_number`, `image_config_id`) are supplied by the
caller of `load_image_config`. The autogenerated field list below is the
source of truth — refer to it for the complete schema.
